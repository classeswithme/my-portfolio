How to Add New Projects Monthly

1. Open this HTML file in any code editor (VS Code, Notepad++, etc.)

2. Find the <div class="upcoming-grid"> section

3. Copy any existing project card (the div with class "glass-card upcoming-card")

4. Paste it before the "Add New Project" button card

5. Update the month badge, title, description, and tags

6. Save the file and commit to GitHub - your site updates automatically!

Pro Tip: You can also add new projects to the "Projects" section the same way. Just duplicate a card and customize it!
